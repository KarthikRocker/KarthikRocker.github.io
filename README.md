## 🛠️ Local Development & Deployment

### Run Locally
Since the portfolio is entirely self-contained within a single file for optimal loading speeds, you can clone it and run it instantly without an active build runner or node dependencies:

```bash
# Clone the repository
git clone [https://github.com/karthikrocker/karthikrocker.github.io.git](https://github.com/karthikrocker/karthikrocker.github.io.git)

# Navigate into the project folder
cd karthikrocker.github.io

# Open index.html directly in your default browser
open index.html
